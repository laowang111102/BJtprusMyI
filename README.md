# Minimal 后台管理系统模板 java193

## 项目概述

Minimal 是一款适用于各类后台管理系统的模板，提供了一套完整且精美的用户界面。

## 技术栈

- 前端：React、Ant Design
- 后端：Node.js、Express.js
- 数据库：MySQL、MongoDB（可选）

## 功能模块

- 用户管理
  - 用户注册、登录、权限管理
- 数据管理
  - 数据展示、编辑、删除
- 系统设置
  - 系统配置、主题切换、权限配置

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：现代浏览器（推荐使用 Chrome、Firefox）
- 后端：Node.js >= 12.0

## 部署步骤

1. 克隆或下载项目代码
2. 安装依赖
   ```bash
   npm install
   ```
3. 运行开发环境
   ```bash
   npm run start:dev
   ```
4. 构建生产环境
   ```bash
   npm run build
   ```
5. 部署到服务器
